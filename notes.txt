Instructions to publish 424apt.com to repository 424apt/424apt.github.io.

Login as davidanderson61 at GitHub website.
  If necessary, regenerate token.
    Click your profile photo, then click Settings.
    In the left sidebar, click <> Developer settings.
    In the left sidebar, click Personal access tokens.
    Select token "For repositories".
    Click Regenerate token.
    Leave page open to copy token later.
Update login keychain if Internet password item named github.com exists.
  Set account to davidanderson61.
  Set password to regenerated token or 424apt's token.
Clone repository 424apt/424apt.github.io.
Open Mobirise.
Select Publish.
  Select Local drive folder.
  Select 424apt.github.io folder containing the repository.
  Select the "Publish" button.
In a Terminal window:
  Stage changes.
  Commit changes.
  Push changes to GitHub.
Publishing is not immediate!

Instructions to copy a repository to another GitHub user.

Login to GitHub as from-user.
git clone --bare https://github.com/from-user/from-repo.git
Login to GitHub as to-user.
  If necessary, regenerate token
    Click your profile photo, then click Settings.
    In the left sidebar, click <> Developer settings.
    In the left sidebar, click Personal access tokens.
    In the left sidebar, click Tokens (classic).
    Select token "For repositories".
    Click Regenerate token.
    Leave page open to copy token later.
  Create a new to-repo repository.
Update login keychain if Internet password item named github.com exists.
  Set account to to-user.
  Set password to regenerated token.
cd from-repo.git
git push --mirror https://github.com/to-user/to-repo.git
cd ..
rm -rf from-repo.git
